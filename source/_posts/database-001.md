# Database 001

## chapter 1：SQL language

e.g.

```SQL
select instructor name
from instructor
where instructor.dept_name = History
```
在 relation model 中，数据以 **retion** 的形式进行表示，而 SQL 语言的作用是用一种标准化语言，对关系数据进行定义、查询、更新和管理

因此我们可以说， **SQL 语言本质是面向关系函数的声明式语言**。

* **relation**  表示一张表
* **tuple** 元组，表示一行
* **attribute** 属性，表中的一列
* **schema** 属性，即表的结构
